# cryptocurrency

fetching API to request a JSON file from https://api.coinmarketcap.com/v1/ticker/?limit=2000 url. Iterate all the available cryptocurrency coins on the cryptocurrency market  and visualize them using HTML, CSS and JavaScript. Your application will provide the following information:
         the number of cryptocurrency coins
         the name of the coins
         the price of the coin
         the rank of the coin
         the change in price of the coin
Features
1.       Display cryptocurrency coins with the required information
2.       Filter the coins by name using input element
Additional Features:
Sort by name
Sort by price
Sort by rank
